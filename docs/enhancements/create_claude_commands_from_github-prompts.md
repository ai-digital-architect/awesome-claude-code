# Skills setup guide

Create a list of claude commands from the awesome github copilot prompts.

# Overview
The github repository https://github.com/github/awesome-copilot contains a set of awesome instructions, chatmodes, prompts and workflows that this project converts into claude commands to be used by claude code.


** Execution Workflow **

- Analyze and review the github repository https://github.com/github/awesome-copilot
- Analyze the https://github.com/github/awesome-copilot/blob/main/docs/README.prompts.md file to understand the available prompts that can  convert to claude commands.
- Gather a list of all prompts from this folder
- Review the ./additional_claude_features.md file to convert the list of prompts to claude commands based on best practices
