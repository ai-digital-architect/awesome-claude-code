# Skills setup guide

Create a list of claude skills from the awesome github copilot instructions.

# Overview
The github repository https://github.com/github/awesome-copilot contains a set of awesome instructions, chatmodes, prompts and workflows that this project converts into claude skills to be used by claude code.


** Execution Workflow **

- Analyze and review the github repository https://github.com/github/awesome-copilot
- Analyze the (https://github.com/github/awesome-copilot/blob/main/docs/README.instructions.md file to understand the available instructions to convert to claude skills
- Review the additional_claude_features.md file to convert https://github.com/github/awesome-copilot/blob/main/instructions/markdown.instructions.md into a claude skill using the recommended best practices for creating claude skills.
